# これは何

AWS のcost 通知をするLambda コードの管理場所です。

## Lambda のコードについて

python 3.10

## デプロイ方法について
環境の準備中です

[GitHub Actions lambda deploy](https://github.com/aws-actions/aws-lambda-deploy?tab=readme-ov-file#usage)

## 環境変数
Line Message API を使っているので、以下の環境変数が必要になります。

```bash
LINEtoken
groupID
```
